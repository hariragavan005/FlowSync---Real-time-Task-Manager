# 📌 Overview

**FlowSync** is a modern real-time collaborative task management platform designed for teams to manage projects, tasks, priorities, and workflows efficiently.

It enables seamless collaboration with real-time updates, role-based access control, activity tracking, and dynamic task organization — all packed inside a responsive and modern user interface.

# ⚙️ Installation Guide

## 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/FlowSync-Real-time-Task-Manager.git
```

---

## 2️⃣ Navigate into Project

```bash
cd FlowSync-Real-time-Task-Manager
```

---

## 3️⃣ Install Dependencies

### Frontend

```bash
cd client
npm install
```

### Backend

```bash
cd server
npm install
```

---

## 4️⃣ Environment Variables

Create `.env` file inside `server` folder:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
CLIENT_URL=http://localhost:5173
```

---

## 5️⃣ Run Project

### Start Backend

```bash
cd server
npm run dev
```

### Start Frontend

```bash
cd client
npm run dev
```
